# apply-label-on-merge

> A GitHub App built with [Probot](https://github.com/probot/probot) that Apply labels to an issue after PR merge.

## Setup

```sh
# Install dependencies
npm install

# Run with hot reload
npm run build:watch

# Compile and run
npm run build
npm run start
```
